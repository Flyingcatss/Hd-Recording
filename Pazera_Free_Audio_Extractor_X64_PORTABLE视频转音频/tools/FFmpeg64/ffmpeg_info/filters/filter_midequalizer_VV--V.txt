Filter midequalizer
  Apply Midway Equalization.
    Inputs:
       #0: in0 (video)
       #1: in1 (video)
    Outputs:
       #0: default (video)
midequalizer AVOptions:
  planes            <int>        ..FV..... set planes (from 0 to 15) (default 15)

This filter has support for timeline through the 'enable' option.
